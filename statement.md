#Problem statement 
Manual ticket booking is time consuming and often leads to errors such as incorrect ticket details, duplication, or missing records. There is a need for a simple computerised system that allows users to book tickets easily and store their bookings safely for future reference.

#Scope of the project 
The project focuses on creating a basic transport ticket booking system using python and text based file storage. the system will allow users to sign up, sign in, view available transports, book tickets and see their booked tickets. this project operates through a command line interface and uses text file to store all records.

#target users 
Useful for beginners trying to learn simple booking systems principals and anyone who want a small simulation of how a ticket booking and storage system operates.

#High level features
User registration and login system. Viewing previously booked tickets. Automatic storage of user, booking, and transport details.
